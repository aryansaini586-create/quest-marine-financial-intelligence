# ⚓ Quest Marine LLC — Financial Intelligence Dashboard

AI-powered financial analysis for Quest Marine LLC — 100% free, no API keys needed.

## Features
- 💬 **Smart Chat** — Ask financial questions, get instant data-driven answers
- 📊 **Dashboard** — KPIs, revenue trends, cost breakdown, geographic analysis
- 🔮 **Scenarios** — Interactive sliders for 3-year projections + break-even
- 📋 **Data Tables** — 7 years of financials with CSV export

## Deploy (Free via GitHub Pages)
1. Fork this repo
2. Go to Settings → Pages → Source: GitHub Actions
3. Push any change to `main` → auto-deploys

## Run Locally
```bash
npm install
npm run dev
```
